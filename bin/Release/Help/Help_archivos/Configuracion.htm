configuración

Aquí se anotan los proxys para el ataque SHOIC HTTP, en el DataGrid que se presenta debe colocar IP del proxy, puerto del mismo y el código que debe ser siempre un número distinto al resto de los códigos, esto es así porque SQLite no autoincrementa como puede hacerlo MySQL o PostgreSQL, ma adelante haré un autoincrement para esta base de datos.

Una vez colocado los proxys, debe presionar en “Guardar”.

Para eliminar un proxy debe posicionarse sobre el mismo y presionar la tecla “supr” o “delete” del teclado, para agregar un proxy, simplemente vaya a la última fila que debe aparecer vacía y comience a completar los campos. Para editar simplemente haga doble click en la celda que desea editar.

Limpiar Autocompletado: Elimina de la base de datos las páginas a las que ha atacado.
